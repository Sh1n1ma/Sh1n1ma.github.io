基于Windows的求生之路2（8人战役）服务器搭建

发表于 2022-02-15 更新于 2023-01-02 分类于 游戏 阅读次数：
本文字数： 1.6k 阅读时长 ≈ 3 分钟 
更新日志：
2022年2月15日（第三部分完成，其余部分不可用。）
1.购买云服务器
2.通过供应商提供的接口远程连接至云服务器
3.在云服务器部署SteamCMD和求生之路2服务端
3.1在云服务器上下载SteamCMD

（1）在云服务器中新建一个文件夹用于存放SteamCMD和steam服务端组件

例如在C盘中新建一个文件夹，名为：steamcmd。

（2）在云服务器上下载SteamCMD（Windows）并解压至步骤（1）的新建文件夹中

官方下载链接：https://steamcdn-a.akamaihd.net/client/installer/steamcmd.zip

3.2在云服务器上使用脚本部署steam服务端和求生之路2服务端

（1）在SteamCMD所在文件夹下（步骤3.1中新建的文件夹）新建一个.bat脚本

可先新建一个.txt(文本文档)，例如：新建一个名为“start”的文本文档。

（2）打开新建的脚本，输入脚本内容。

　　脚本内容如下（复制粘贴即可）：

start steamcmd.exe +login anonymous +force_install_dir 14d2_sv +app_update 222860 validate +quit

　　输入内容后，关闭文本文档，将其后缀修改为.bat。例如，将start.txt重命名为start.bat。

（3）双击运行脚本，等待steam服务端和求生之路2服务端下载完成。若下载成功，文件夹中将出现名为l4d2_sv的子文件夹，其中为求生之路2服务端的组件。

4.进行求生之路2服务端初始化配置

​ 服务器配置文件位置：\l4d2_sv\left4dead2\cfg\server.cfg

　　打开l4d2_sv文件夹中的left4dead2文件夹，再打开cfg文件夹，该文件夹中应有一个名为server.cfg的配置文件（若不存在可以自己创建一个）。用记事本打开该配置文件，输入以下内容（复制粘贴即可）：

sv_tags "hidden"

//player number
l4d_maxplayers 8
sv_maxplayers 8
sv_visiblenmaxplayers 8
sv__removehumanlimit "1"

//game mod
sm_cvar mp_gamemode "coop"

// lobby
sv_allow_lobby_connect_only 0

sv_lan 0
sv_consistency 0

5.启动游戏服务器

​ 使用脚本启动服务器，在游戏服务端根目录下新建文本文档，输入以下内容：

start srcds .exe -console -game left4dead2 -insecure +map c1m2_streets -condebug +exec server.cfg -nomaster

保存并退出，将文件的后缀修改为.bat。双击文件即可启动服务器。在玩家本地游戏内打开控制台输入connect 服务器ip地址即可连接至服务器进行游玩。

6.插件安装（选）

推荐望夜整合包、豆瓣酱整合包。

7.管理员权限配置

一般在安装了插件后，需要进行管理员权限配置。以sourcemod的管理员权限配置为例，权限配置文件应在\14d2_sv\left4dead2\addons\sourcemod\configs路径下，文件名为admins_simple.ini。

在文件最后进行管理员权限的添加。例如，在文件的最后输入：

"STEAM_0:1:209066414" "99:z"

前半部分代表管理员的steam id（可在网站https://steamid.io/lookup/进行查询），后半部分代表赋予该玩家管理员权限。

最后保存文件，管理员权限配置结束。（望夜插件的常用的管理员命令有：!admin !rygive）
