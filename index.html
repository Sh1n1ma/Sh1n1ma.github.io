2023/1/29

发表于 2023-01-29
本文字数： 742 阅读时长 ≈ 1 分钟

今天老爸找我，在家中一位老人离世的背景下，谈关于人生进程，关于死亡，关于生活，在此做一些记录。

核心的，重要的思想是：

生命、亲情都是宝贵的，应当加以珍惜。

关于生命，虚度光阴也许并不是一个很好的生活方式。因为有许多人，花费金钱，消耗精力，但是他们再也跑不赢时间了，离开了我们现在的世界，如果我们这样的年轻人就这样让时光从指尖流走，是否有些不值得呢？

关于亲人，尤其是长辈，对于他们而言，时间、家人的陪伴或许更加宝贵，我们应当加以关怀。若有不幸患病的情况，更应加以关心与爱护，疏解病人的情绪，与病人沟通，帮助病人对抗病魔，实现一些心愿，都是晚辈在有条件的情况下应当做到的。

我们应当尊重生命的自然规律，在生活中发挥人的主观能动性。乐观一些，积极一些。

生老病死，对于所有人而言暂且是公平的，人生的尽头不过是“尘归尘。土归土。”我们应当认识到生命进程的客观规律。生活中，每个人有每个人的苦恼，都会面对困难的局面，在面对生活时，也应当发挥自己的能量，积极地生活，争取让快乐多一点，让烦恼少一些。人到世界上走一遭，并不容易，意外也随时可能发生，我们可以寻找一些自己的热爱所在，或许这就是人生的意义。（世界上只有一种英雄主义，在认清生活真相之后依然热爱。）

作为年轻一代，对未来要有一些想法。有了目标之后，需要积极地，努力地去完成。若能够有所成就，达成目标，是最好的结局。如果是三番五次的努力，还是难以实现，那就需要审视局面，或许可以选择放弃认输，挫败感应当是存在的，在努力过后也可以释怀，生活不可能永远遂人愿，只需尽力而为！有想法就尝试着实现，无论在何种境地之下，家人永远是你最坚强的后盾，支持、鼓励、帮助你。（just do it！）

关于友情，有几位知心朋友，生活会更美好的！此外，对他人的要求也不要太高。

腾讯云服务器部署RSSHub服务小记

发表于 2022-09-09
本文字数： 669 阅读时长 ≈ 1 分钟

云服务器镜像: CentOS7.6-Docker20

RSSHub官方部署教程，https://docs.rsshub.app/install/#docker-jing-xiang

本次部署通过Docker Compose 部署.

1.安装docker

首先安装docker，本次部署使用腾讯云预置docker镜像的服务器进行操作。

1.1 安装docker compose

下载 docker compose(通过kgithub镜像下载)

$ sudo curl -L "https://kgithub.com/docker/compose/releases/download/v2.10.2/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

进行本地环境操作（待补充）

$ sudo chmod +x /usr/local/bin/docker-compose
$ sudo ln -s /usr/local/bin/docker-compose /usr/bin/docker-compose

下载docker-compose.yml，也就是RSSHub服务的配置文件。

$ wget https://raw.githubusercontent.com/DIYgod/RSSHub/master/docker-compose.yml
1.2 创建 volume 持久化 Redis 缓存
$ docker volume create redis-data
1.3 启动服务
$ docker-compose up -d
